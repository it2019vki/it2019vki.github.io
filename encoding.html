По умолчанию блокнот сохраняет файл в кодировке windows-1251. Обратите внимание, что в примерах ниже указана другая кодировка, utf-8. Блокнот может сохранить и в utf-8, если явно выбрать эту кодировку при сохранении файла.
Для указания кодировки используется тэг meta:
<meta http-equiv="Content-Type" content="text/html; charset=utf-8">
Для пятой версии HTML можно использовать сокращённый вариант:
<meta charset="utf-8">

Приближается один из важнейших дней учебного года - контрольная неделя. Для получения оценки за контрольную неделю необходимо создать на своём сайте две странички с информацией из приложенного pdf. Первая страничка сохраняет форматирование из pdf файла, а вторая должна иметь свой собственный уникальный стиль форматирования. Две созданные html странички должны отличаться только именем и подгружаемым css файлом.
